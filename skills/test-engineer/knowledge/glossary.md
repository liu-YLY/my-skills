# 术语索引(精简)

> **何时阅读**:对术语含义有歧义时再看;否则可跳过。
> **覆盖范围**:P0-P3 / 冒烟 / 回归 / 等价类 / 边界值 等 5 个核心术语。
> **可跳过条件**:已经熟悉测试基础术语。

与 [test-standards.md](test-standards.md) 配合使用;详细分级与流程以该文件为准。

| 术语 | 含义 |
|------|------|
| **P0–P3** | 测试优先级：P0 最关键路径/阻塞级，P1 主流程与重要异常，P2 次要/边界，P3 体验与极端边界 |
| **冒烟测试** | 最小集验证构建可测，失败则阻塞更细测试 |
| **回归测试** | 变更后重跑相关用例，确认未引入新缺陷 |
| **等价类** | 将输入划分为等价集合，每类选代表值测试 |
| **边界值** | 在等价类边界及边界两侧取值的测试方法 |
